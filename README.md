# More about this repository - Think First Lit
**A system for producing trustworthy AI-assisted literature reviews.**

Run the pipeline and get:
- **Annotated literature review** (`.docx` + `.md`)  
- **Slide deck outline** (`.pptx`)  
- **Auditable research log** (searches, sources, gaps)  

All grounded in real, retrievable academic sources.

## How It Works

A structured, multi-phase workflow based on skills developed by Consensus:
1. **Recon** — Map the field and terminology  
2. **Frame** — Apply a research model (PICO / SPIDER / decomposition)  
3. **Search** — Execute systematic, rate-limited queries (Consensus)  
4. **Analyze** — Tag, categorize, and summarize papers  
5. **Produce** — Generate publication-ready outputs  

---
## Design goals for this pipeline

**No hallucinated citations**  
Only papers retrieved during the session are included.

**Built-in audit trail**  
Every search, failure, and gap is logged.

**Structured for practitioners**  
All findings organized into:
- Methods  
- Impact  
- Governance  

**Metadata you can trust**  
Each paper includes links, citations, tags, and confidence flags.

**Amplificaiton**  
Skills are included to ease the burden of sharing the lit review with collegues online

---

## When to Use It

Use this when you need:
- A **structured literature review**, not a summary   
- Citations you can verify (using Google Scholar)
- Outputs you can share or publish  

Skip it when you need:
- A quick answer  
- A single paper  
- A different domain
